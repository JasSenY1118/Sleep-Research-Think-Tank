Preprocessing pipeline

Intended static pipeline before model-ready data exists.

  1. Load EDF signal files
  2. Load hypnogram annotations
  3. Align signal with annotations
  4. Segment into 30-second epochs
  5. Map sleep-stage labels
  6. Filter EEG signal
  7. Normalise each epoch
  8. Exclude or separately handle movement/unknown labels
  9. Save processed arrays as .npz or .pt
  10. Split by subject into train, validation, and test sets

Future model roadmap

Candidate model directions for later phases.

CNN-LSTM

Intended use: model temporal context across adjacent epochs.

Expected advantage: captures sequence structure beyond one epoch.

Current status: future phase.

Transformer

Intended use: explore attention-based sequence modelling.

Expected advantage: flexible long-range temporal context.

Current status: future phase.

Expected advantage: richer physiological context.

Current status: future phase.

Expected advantage: stricter estimate of transfer to new people.

Evaluation metrics

Short definitions for future reporting.

Accuracy

Overall proportion of correctly classified epochs.

Macro-F1

Average F1 score across classes, treating classes evenly.

Balanced accuracy

Average recall across classes, useful when classes are uneven.

Confusion matrix

Table showing predicted versus reference labels.

Per-class recall

How often each sleep stage is correctly recovered.
